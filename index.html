Trash - A Waste Management Initiative

The Trash App is a comprehensive waste management solution designed to promote recycling by enabling users to sell their waste for responsible disposal. Built using React Native for the user app, React.js for the admin panel, and Node.js with Express and MongoDB for the backend, the app provides a seamless experience for individuals, schools, colleges, offices, and factories. It features Google-based signup with Clerk, waste estimation, and an integrated social media platform for community engagement. Users can access chat support with an AI bot and admin, educational YouTube videos, and a review system powered by Gemini AI for feedback analysis. This app streamlines waste management while fostering environmental responsibility.

Book My Movie

The Movie Ticket Booking Website is a responsive web application developed using both Tailwind CSS and Bootstrap to provide a seamless and visually appealing user experience. The platform allows users to explore movies, view details such as posters, trailers, and show timings, and proceed with ticket reservations through an intuitive booking flow. By combining the utility-first customization of Tailwind with the prebuilt responsive components of Bootstrap, the website achieves a balance between flexibility and consistency in design. The project showcases a modern, user-friendly interface with smooth navigation, making the process of booking movie tickets simple and engaging across all devices.
